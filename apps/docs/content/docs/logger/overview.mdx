`@okfetch/logger` is a plugin for `@okfetch/fetch`, not a standalone client. Use it when you already make requests through `okfetch` and want logging in the request lifecycle.

## Install

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm">

```bash
npm install @okfetch/logger @okfetch/fetch pino
```

  </Tab>
  <Tab value="pnpm">

```bash
pnpm add @okfetch/logger @okfetch/fetch pino
```

  </Tab>
  <Tab value="yarn">

```bash
yarn add @okfetch/logger @okfetch/fetch pino
```

  </Tab>
  <Tab value="bun">

```bash
bun add @okfetch/logger @okfetch/fetch pino
```

  </Tab>
</Tabs>

## Basic usage

```ts twoslash
import { okfetch } from "@okfetch/fetch";
import { logger } from "@okfetch/logger";

const result = await okfetch("https://example.com/health", {
  plugins: [logger()],
});
```

## Log parsed success data

```ts twoslash
import { okfetch } from "@okfetch/fetch";
import { logger } from "@okfetch/logger";

const result = await okfetch("https://example.com/health", {
  plugins: [
    logger({
      logDataOnSuccess: true,
    }),
  ],
});
```

## Pass `pinoOptions`

```ts twoslash
import { okfetch } from "@okfetch/fetch";
import { logger } from "@okfetch/logger";

const result = await okfetch("https://example.com/health", {
  plugins: [
    logger({
      pinoOptions: {
        level: "debug",
      },
    }),
  ],
});
```
