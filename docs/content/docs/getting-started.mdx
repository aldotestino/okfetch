Start by making one request with `@okfetch/fetch`. If your API surface grows, move the repeated request shapes into `@okfetch/api`. If you want logs, add `@okfetch/logger` on top of `@okfetch/fetch`.

## First request

This is the baseline pattern used across the docs: call `okfetch`, validate the result, then branch explicitly on success and failure.

```ts twoslash
import { okfetch } from "@okfetch/fetch";
import { z } from "zod/v4";

const userSchema = z.object({
  email: z.string().email(),
  id: z.number(),
  name: z.string(),
});

const result = await okfetch("/users/:id", {
  baseURL: "https://api.example.com",
  params: { id: 1 },
  outputSchema: userSchema,
});

result.match({
  err: (error) => console.error(error._tag, error.message),
  ok: (user) => console.log(user.name),
});
```
